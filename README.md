# Star Trader

A web-based space trading simulator built with TypeScript, Vite, and Phaser.

## Getting started

```bash
npm install
npm run dev
```

## Stack

- [Vite](https://vite.dev/) — build tooling and dev server
- [TypeScript](https://www.typescriptlang.org/)
- [Phaser](https://phaser.io/) — game engine/renderer
